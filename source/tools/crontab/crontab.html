crontab表达式说明：

  *    *    *    *    *    [user]   [command]
  -    -    -    -    -
  |    |    |    |    |
  |    |    |    |    |
  |    |    |    |    +----- 星期 (0 - 7) (Sunday=0 or 7)
  |    |    |    +---------- 月 (1 - 12)
  |    |    +--------------- 日 (1 - 31)
  |    +-------------------- 时 (0 - 23)
  +------------------------- 分 (0 - 59)

crontab命令实例

0 0 L * *        #每月的最后1天执行
* * * * *        #每1分钟执行一次
*/10 * * * *     #每隔10分钟执行一次
0 */1 * * *      #每隔1小时执行一次
30 21 * * *      #每晚的21:30执行
3,15 * * * *     #每小时的第3和第15分钟执行
3,15 8-11 * * *  #在上午8点到11点的第3和第15分钟执行

cron服务操作命令
service crond start    #启动服务
service crond stop     #关闭服务
service crond restart  #重启服务
service crond reload   #重新载入配置
service crond status   #查看服务状态

crontab -l   #查看crontab
crontab -e   #编辑crontab
crontab -r   #删除crontab
